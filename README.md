# ✈️ Flight Price Prediction - EDA & Feature Engineering

## 📌 Overview
This project focuses on analyzing a flight price dataset using exploratory data analysis (EDA) and feature engineering techniques.

The goal is to clean the dataset, extract meaningful features, and understand factors affecting ticket prices.

---

## 📊 Key Steps
- Data Cleaning (handling missing values, correcting data types)
- Feature Engineering (date, time, duration extraction)
- Exploratory Data Analysis (EDA)
- One-Hot Encoding for categorical variables

---

## 📈 Insights
- Airline choice significantly impacts ticket prices
- Number of stops affects pricing (non-stop flights are usually more expensive)
- Duration of flight is an important factor
